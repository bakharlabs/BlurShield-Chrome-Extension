Toolbar Guide

Learn how to use the floating toolbar that appears when you click the extension icon.

Click Blur Mode

Click on any element (images, divs, buttons) to blur them instantly. Great for hiding sensitive content quickly.

Draw Areas Mode

Click and drag to create custom rectangular blur areas. Perfect for blurring specific sections or multiple elements at once.

Text Selection Mode

Select any text on the page to blur it. Ideal for hiding sensitive text content, passwords, or personal information.

Eraser Mode

Remove existing blurs by clicking on them. Use this to undo specific blurs without clearing everything.

Show/Hide Blurs

Toggle all blurs on the page on/off without removing them permanently. Great for temporarily viewing content.
